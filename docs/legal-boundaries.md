# 使用边界与发布说明

MusicTool 的公开定位是：读取公开歌单元数据，辅助用户匹配自有、已授权或平台允许保存的音源，并整理本地音乐库。

## 本项目不包含的内容

- 不提供、不托管、不分发音乐、歌词、封面或其他受版权保护的内容。
- 不提交真实 Cookie、账号材料、访问日志、pid、临时 manifest、真实下载报告。
- 不提供绕过 DRM、付费内容、会员内容、私有内容或访问控制的能力。
- 不把 QQ 音乐歌单元数据读取误写成获取平台音源。

## 用户需要自行确认的事项

- 候选音源是否有权保存或离线使用。
- 目标平台条款是否允许自动化访问、下载或 Cookie 使用。
- 本地 Cookie 文件是否仅在自己的设备上使用，且不会被提交到仓库。
- 生成的歌词、封面和音频是否只用于个人合法场景。

## 公开仓库检查

公开前应确认：

- `git status --short` 中没有音频、歌词、Cookie、日志、pid、输出目录。
- `examples/` 只包含虚构数据。
- README 使用保守定位，不承诺下载任何平台的受保护内容。
- 运行产物已移动到仓库外。
